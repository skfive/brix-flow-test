3. 셀 시각 규칙 — 머리 4방향 · 몸 · 꼬리 · 먹이 · 빈 셀
머리 · up
눈이 위쪽
머리 · down
눈이 아래쪽
머리 · left
눈이 왼쪽
머리 · right
눈이 오른쪽
몸(body)
초록 그라데이션 + 마디선
꼬리(tail)
몸보다 살짝 작게
먹이(food)
앰버 원형 + 꼭지 + 글로우
빈 셀(empty)
보드 배경 + 격자선만

색맹 대응: 머리 vs 몸은 명도 차 + 눈 형태, 먹이 vs 뱀은 색상 대비 + 원형(꼭지) 으로 색 인지 없이도 구분됩니다.
