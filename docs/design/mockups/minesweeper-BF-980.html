⑤ 반응형 보드 스케일 규칙 — §4.4
난이도별 셀 크기(--cell-size) & 좁은 뷰포트 대응:
· 초급 9×9 — 기본 34px / clamp(30px,9.5vw,34px) → ~324px, 대부분 화면 스크롤 없이 수용
· 중급 16×16 — 기본 30px / clamp(22px,5.6vw,30px) → ~496px, 태블릿까지 수용, 소형은 셀 축소
· 고급 30×16 — 기본 26px / clamp(20px,3vw,26px) → ~810px → 뷰포트 초과 시 .board-scroll 가로 스크롤(셀 하한 20px 유지, 판독성/터치 타깃 우선)
· breakpoint — ≥768px: 기본 셀 / 480–767px: clamp 중간값 / <480px: 상태바 2줄 wrap + 고급 가로 스크롤 활성
셀 내부 숫자·글리프 폰트는 셀 크기에 비례(em/clamp) 축소. 셀을 20px 미만으로 무리하게 줄이지 않음.
접근성 시각 규칙 요약 — 포커스 링 2px var(--color-focus-ring)(outline:none 단독 금지) · roving tabindex(활성 셀 1개만 tabindex=0) · 종료 시 disabled 미부여(포커스 유지) · prefers-reduced-motion에서 전환 애니메이션 무효 · 숫자 1~8 AA 대비 + 글리프 병행(색맹 대응).
