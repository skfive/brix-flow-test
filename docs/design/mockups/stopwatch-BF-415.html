4. 키보드 단축키 (§6.2)

전역 단축키

키동작조건
Space 시작 ↔ 정지 토글 전역
L / l 랩 추가 running 중에만
R / r 리셋 stopped / max-cap
Esc 리셋 (R 보조) timer SPA 와 일관성
Tab / Shift+Tab topbar 토글 → 시작 → 정지 → 랩 → 리셋 → 랩 카드 [지우기] 순환 전역
Enter 현재 focused 버튼 클릭 버튼 focus 시 (브라우저 기본)

5. focus-visible 정적 표현

focus-visible · primary button LIGHT
