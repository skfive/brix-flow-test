전경 / 배경 쌍 라이트 다크 기준

힌트(muted) / canvas ⚠️ ≈ 2.6:1 (장식 보조) ≈ 4.2:1 통과 일반 4.5 — 아래 주석

▸ 계약 고정(변경 금지): 값 표시는 <output id="counter-value" aria-live="polite">, 버튼은 네이티브 <button> 3개(btn-increment / btn-decrement / btn-reset), DOM/Tab/시각 순서 모두 +1 → -1 → 초기화 (기획 §4.2 · 명세 §4.1).
▸ 원본 대비 유일한 시각 차이: <title>·<h1> 식별 문자열뿐. 카드·값·버튼·힌트 토큰은 /demo/counter(BF-862)와 1:1 동일 — 재사용 검증의 시각적 근거.
▸ 포커스 가시성(본 task 핵심): 프레임 ③④⑤ 는 +1/-1/초기화 각각의 :focus-visible 링을 시연. variant 색과 무관하게 accent outline + focus-ring box-shadow 로 통일. outline:none 단독 사용 금지.
▸ 접근성 토큰: 최소 터치 타깃 48px · 색 비의존(텍스트 라벨 + <kbd>) · aria-live announce · prefers-reduced-motion 시 전환 제거 · 라이트/다크 자동 전환.
▸ ⚠️ 힌트 대비 주석: 라이트 힌트(muted ≈2.6:1)는 원본 토큰 승계분이라 임의 변경 시 원본 회귀 위험. 장식성 보조 정보(핵심 조작은 버튼·aria-live로 완결)이며, 개선 여부는 원본 오너/운영자 판단 영역(명세 §2.3 · §10).
