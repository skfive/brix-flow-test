mockup 사용 가이드

  • 본 HTML 은 시각 시뮬레이션 전용. dev 산출물 X. 픽셀 일치 의무 없음.
  • 실 동작 (입력·평가·키 바인딩) 은 main.js / calc.js 에서 구현 — 본 mockup 의 키는 정적 (onclick 없음).
  • focus-visible / active 시각은 .is-focus / .is-active 클래스로 강제 표현. 실 구현에서는 브라우저 기본 :focus-visible / :active 의사 클래스로 처리.
  • 연산자 키 라벨은 × ÷ − 로 표시 (입력값은 * / -). aria-label 은 한국어("곱하기" 등).
  • 전체 정량 치수와 토큰 매핑은 docs/design/calculator-BF-410.md §7.7 정량 일치 표를 source of truth 로 참조.
