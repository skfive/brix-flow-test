3. 카드 상태 레전드 (§5.1)
◆🍇

hidden — 뒷면

통일 문양 ◆, 그라디언트. 클릭/키 가능, hover 시 상승. aria: "카드 N, 뒤집기 전"

◆🍉

revealed — 앞면 공개

밝은 면 + accent 테두리. 하단 쌍별 색 바. 재클릭 no-op. aria: "카드 N, 수박"

◆🍎

matched — 짝 확정

성공 글로우 + 초록 테두리, 고정. 상호작용 없음(aria-disabled, 포커스 유지)

◆🍒

mismatch — 불일치 순간

경고 테두리 + shake(~400ms). 직후 hidden 복귀(지연 800ms). 색+모션 이중 표현
