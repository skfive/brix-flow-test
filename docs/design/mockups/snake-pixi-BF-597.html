07 · pixi.js Scene Graph Z-Order 구조도
PIXI.Application.stage 계층 구조
stage (PIXI.Application.stage)
  ├── [0] backgroundContainer ← Graphics: 배경(0x0d0d0d) + 격자라인(0xffffff, alpha 0.04) — 정적, resize 시만 재생성
  ├── [1] foodContainer ← 먹이 원(방사형 그라데이션) + 글로우 ring + 배수 텍스트. food=null 시 visible=false
  ├── [2] itemContainer ← 아이템 원 + 아이콘 Graphics + 만료 alpha 페이드. item=null 시 visible=false
  ├── [3] extraCpuContainer ← extraCpus 배열 (0~5마리). 빈 배열 시 removeChildren
  ├── [4] cpuContainer ← CPU 지렁이. cpu=[] 시 removeChildren
  └── [5] playerContainer ← 플레이어 지렁이 (최상위 캔버스 레이어)

DOM 오버레이 (canvas 위, z-index 기반 — 불변)
  ├── #effect-layer ← 파티클 이펙트 (DOM, pixi 무관)
  ├── #hud ← 점수 HUD (PLAYER/CPU/Best)
  ├── #hud-status-panel ← 길이/속도/남은시간 상태 패널
  ├── #item-slot-hud ← 보유 아이템 슬롯 [Z]
  ├── #buff-bar ← 활성 버프 상태
  ├── #multiplier-stats ← 배수 통계 (좌상단)
  ├── #competition-legend ← 범례 (좌하단)
  ├── #gameover-overlay ← 게임오버 모달 (DOM)
  ├── #paused-overlay ← 일시정지 모달 (DOM)
  └── #settings-modal ← 설정 모달 (DOM)
