인터랙션 시각 가이드

  • 카드 hover lift: transform: translateY(-4px) + box-shadow 강화 + border-color 강화, 모두 200ms cubic-bezier(0.2, 0, 0, 1) 일치 (AC 명시). 본 mockup HTML 을 브라우저로 직접 열어 카드 hover 시 확인 가능.
  • 페이지 진입 stagger fade-in: 카드 6장이 80ms 간격 (0 → 400ms) 으로 opacity 0 → 1, translateY 8px → 0, 각각 280ms cubic-bezier(0.2, 0, 0, 1.2). 본 mockup 페이지 로드 시 자동 재생.
  • 이모지 halo: data-state attribute (sunny / cloudy / rainy / snowy / thunder / windy) 에 따라 이모지 뒤 원형 배경 색이 자동 매핑. 카드 본체 톤을 흩뜨리지 않도록 16% 이하 opacity 사용.
  • 다크 토큰 페어: --color-bg-canvas: #0d1117 (GitHub Dark) + --color-accent: #38bdf8 (Tailwind sky-400) — AC 직접 명시. 온도 단위 °C 가 accent 색으로 표시 (시각 anchor).
  • 반응형 grid: 본 mockup 의 다크/라이트 패널은 desktop 3 열 고정 (1240px viewport), 하단 mobile 패널은 420px 폭 강제 → 1 열. tablet 2 열은 브라우저 폭을 600 ~ 959 사이로 줄여 확인.
  • file:// 호환: 본 mockup HTML 은 외부 CDN · SVG · 웹폰트 · fetch() · import 0 건. file:///…/weather-BF-435.html 로 직접 열어도 동작. DevTools console 의 에러 / Network 외부 호출도 0 건이어야 합니다.
  • prefers-reduced-motion: OS / 브라우저 설정에서 모션 감소 활성 시 모든 animation · transition · hover transform 비활성 — 카드 즉시 표시.
