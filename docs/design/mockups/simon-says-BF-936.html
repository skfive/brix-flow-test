BF-936 · designer [이디자인] · vanilla-static · 명세: docs/design/simon-says-BF-936.md

정적 mockup 이라 실시간 시퀀스 재생은 재현되지 않습니다. 대신 게임 상태별 스냅샷을 나란히 배치하여 상태 전이(대기 → 시퀀스 재생 → 당신 차례 → 실패)를 한눈에 비교할 수 있게 했습니다. 각 패드는 색 + 위치(2×2 사분면) + 텍스트 라벨 + 숫자키로 삼중 식별되며, 실제 <button aria-label> 마크업 + :focus-visible 링을 포함해 키보드 접근성을 검증할 수 있습니다.

1. 시작 → 2. 다시하기 → 3. Green(1) → 4. Red(2) → 5. Yellow(3) → 6. Blue(4)
컨트롤을 먼저, 이후 패드를 읽기 순서(↖↗↙↘)로. 숫자키 1·2·3·4 는 포커스와 무관하게 전역 입력. Enter/Space 로 포커스된 요소 활성. 시퀀스 재생 중 패드는 aria-disabled 로 입력 차단.
