② 결함 B — 오버레이 타이틀 포커스 표시 (before / after) — §5.1

.overlay__title 이 .focus() 를 받아도 outline:none 만 있어 키보드 사용자에게 안 보이던 문제 → :focus-visible 링 부여.

✕ before (현재 결함)

일시정지

포커스는 있으나 시각 표시 없음 — 키보드 사용자 혼란

✓ after (수정본)

일시정지

.overlay__title:focus-visible 링 표시 — 어디에 포커스가 있는지 명확

CSS: .overlay__title:focus-visible { outline: 2px solid var(--color-accent); outline-offset: 4px; box-shadow: 0 0 0 4px var(--color-focus-ring); } — base outline:none 은 유지(비키보드 포커스 노이즈 방지).

③ 결함 A — 재개/시작/재시작 후 포커스 목적지 — §5.2

오버레이가 hidden 되는 순간 포커스가 hidden 요소에 고립되던 문제 → 그 시점 보이는 요소로 이동. 두 후보 비교(최종 선택은 dev).
