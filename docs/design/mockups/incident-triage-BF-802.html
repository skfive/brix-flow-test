4. 버튼 상태 — 기본 / hover / disabled / 복사됨

disabled 복사 버튼은 opacity 로 흐리게 처리하지 않습니다 — 대비가 깨지기 때문입니다. 명시적 회색 토큰으로 4.52:1 을 유지합니다. "복사됨" 은 클릭 후 2초 뒤 원래 라벨로 자동 복귀합니다 (기획 §4.3 고정 타이밍).

5. 포커스 링 — 키보드 전용 사용자 (AC-08)

Tab 순서는 DOM 순서 그대로: 영향도 3 → 긴급도 3 → 요약 복사 → 초기화. 모든 포커스 가능 요소에 outline: 2px solid #3563E9; outline-offset: 2px 를 적용하며 outline: none 단독 사용은 금지합니다. 아래는 포커스 링이 적용된 모습입니다 (실제로 Tab 키를 눌러 확인해 보세요).

6. 모바일 프리뷰 (360px) — 반응형 (AC-10)

≤480px 에서 영향도/긴급도 그룹이 세로 스택으로 재배치되고, 버튼은 100% 폭으로 늘어나 터치 타깃(44px)을 확보합니다. 320px 최소 폭에서도 가로 스크롤이 발생하지 않습니다.
