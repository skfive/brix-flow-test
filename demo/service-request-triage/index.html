서비스 요청 분류 보드

요청마다 유형(Type) × 영향도(Impact) 를 선택하면 P1~P4 우선순위가 즉시 계산되어 해당 우선순위 컬럼으로 분류됩니다. 미선택 요청은 미분류 컬럼에 모입니다. (로컬 상태 전용 데모)

🐛유형 범주형 — 아이콘+라벨 태그 영향도 — 서열형 4단계 강도 미터 P1 우선순위 — 시맨틱 색상 배지 + 컬럼

참조 — 유형(Type) × 영향도(Impact) → P1~P4
